## 四、ionic 的安装使用

①安装指定版本命令 npm install -g ionic@3.12.0

②安装最新版本 npm install -g ionic@latest

3、创建项目 ionic start myApp(项目名称) tabs（blank、tabs、sidemenu）；

4、cd 到项目目录

5、运行 ionic serve
